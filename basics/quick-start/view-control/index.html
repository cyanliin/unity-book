# 調整編輯視角

在編輯 3D 場景時，會時常需要旋轉視角，下面這三個是最常使用的操作。搭配上面改變物件的方法，熟練以後，就可以在 3D 環境中自由地編輯。

操作效果
滑鼠滾輪Zoom In / Zoom Out
Option + 滑鼠右鍵旋轉編輯視角
F聚焦到選取的物件
