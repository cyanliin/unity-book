# Unity 遊戲開發聖經

# 簡介

本教學嘗試使用三種學習主題互相參照的方式來進行，達成容易上手，又能有效學習扎實知識。每個主題的學習方向皆不相同。建議同時互相參照。

# Unity 基礎

逐項講解 Unity 功能，提供系統性的詳細解說，讓知識更扎實。

# 遊戲設計模式

學習實用的遊戲程式設計通用技巧，可應用到其他遊戲引擎。

# 遊戲專案

以多個經典的實作範例，提供最直觀的學習案例，快速學習各種類型遊戲的製作方法。
